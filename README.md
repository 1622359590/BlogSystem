## 私人博客系统

BLOG 是学习 JavaWeb 开发4个月以来的总结实践项目，使用 SSM（Spring、SpringMVC、MyBatis）框架，
MVC 三层结构，Shiro 身份，权限管理，Lucene 全文检索引擎工具的个人博文系统。
