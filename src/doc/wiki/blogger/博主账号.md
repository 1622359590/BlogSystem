- 返回格式：json
- 请求方式：post
- 请求参数说明：

## 2. 更新账号
- 接口地址：http://localhost:8080/blogger/1 （数字1为博主id）
- 返回格式：json
- 请求方式：put
- 前提条件：博主需登录
- 请求参数说明：如果一个参数都不指定，则不执行更新
<table>
<tr>
<th>名称</th>
<th>类型</th>
<th>必填</th>
<th>说明</th>
<th>默认</th>
</tr>
<tr>
<td>username</td>
<td>string</td>
<td>否</td>
<td>用户名</td>
<td>不修改</td>
</tr>
<tr>
<td>password</td>
<td>string</td>
<td>否</td>
<td>密码</td>
<td>不修改</td>
</tr>
</table>

- 返回值：
更新成功时无返回值

## 3. 注销账号
- 接口地址：http://localhost:8080/blogger/1 （数字1为博主id）
- 返回格式：json
- 请求方式：delete
- 前提条件：博主需登录
- 返回值：
注销成功时无返回值
